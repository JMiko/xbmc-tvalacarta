Para crear el instalador en Windows

0) Tiene que estar en el directorio c:\py2exe\tv3, el script de inno setup depende de ello

1) Sobre el directorio "tv3" ejecutar py2exe -> crea la distribución en ./dist

	python setup_win.py py2exe

2) Copiar rtmpdump.exe a ./dist, junto al recién creado descargar.exe

3) Ejecutar setup.iss con Inno Setup, se abrirá el editor

4) Incrementar el nº de versión

5) Darle a Build / Compile

6) El instalador se crea en ./Output
